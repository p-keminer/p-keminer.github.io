# Docs

Dokumentationszentrale fuer Architektur, Asset-Workflow und Sprintplanung.

## Inhalt

- `architecture/` beschreibt Schichten, Datenfluss und Kerntrennungen.
- `assets/` sammelt Blender- und GLB-Konventionen.
- `sprints/` beschreibt Umsetzungsreihenfolge und kurzzyklische Arbeitspakete.

## Pflege

Wenn sich Architektur, Asset-Namen, Workflows oder Sprintziele aendern, muessen die betroffenen Unterdokumente und die Root-Dokumentation synchron bleiben.
