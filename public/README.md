# Public Assets

Statische Laufzeit-Assets fuer Modelle, Texturen und spaetere Medien.

## Struktur

- `models/` fuer GLB-Dateien
- `textures/` fuer optionale Brett- und Oberflaechentexturen

## Regel

Dateinamen und Asset-Kontrakte muessen dokumentiert bleiben, damit Loader und Dokumentation nicht auseinanderlaufen.
